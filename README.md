# Etch-a-Sketch

A browser-based drawing application built with JavaScript that allows users to draw on a grid by clicking over squares.

## Features
- Dynamic grid generation
- Adjustable grid size
- drawing effect
- Clear and reset functionality
- Responsive grid layout

## Technologies Used
- HTML
- CSS
- JavaScript
- DOM manipulation

## How It Works
The program generates a grid of square div elements inside a container.  
When the user hovers over a square, JavaScript modifies the square’s background color to simulate drawing.

Users can also regenerate the grid with a new size, allowing for different drawing resolutions.
